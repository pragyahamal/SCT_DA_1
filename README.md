# Superstore Sales Dashboard

## Objective
Analyze sales data using Microsoft Excel and create an interactive dashboard using Pivot Tables and Charts.

## Features
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Pivot Tables
- Pivot Charts
- Dashboard Visualization
- Conditional Formatting

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- GitHub
